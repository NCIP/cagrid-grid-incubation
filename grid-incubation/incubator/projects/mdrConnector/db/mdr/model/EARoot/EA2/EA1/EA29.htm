A Terminological Entry applies to an Administered Item in a particular Context. It provides a grouping of Designations and Definitions partitioned into Language Sections, allowing the Administered Item to be named and defined within the Context in multiple languages.

An Administered Item may have one or more Terminological Entries, each in a particular Context. Each Terminological Entry contains one or more Language Sections as represented by the relationship terminological entry languages in Figure 6.
