PostgreSQL Schema README

The contents of this directory are used to create a PostgreSQL CSM database. It is also used to create a configuration file that describes the database to applications that use the database.

Here are descriptions of the files in this directory:

AuthSchemaPostgres.sql
An SQL script for creating the CSM tables and their related constraints and indexes.

createDatabase.sql
A script for creating the database.

DataPrimingPostgres.sql
A script to populate tables with initial reference data.

db_build.xml
An ant script to run the other scripts. This is intended to be included by another ant script and not run stand-alone.

GrantPostgres.sql
A script to grant access to CSM tables.

README.html
This file

UserProtectionElements.sql
A script to create a view of the CSM tables that greatly simplifies determining what a user is permitted to do.
