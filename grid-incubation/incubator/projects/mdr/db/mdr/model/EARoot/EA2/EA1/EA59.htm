The composite datatype Language Identification serves as an identifier for a language. It is used in:
* the Registration Authority class to identify the default language(s) of the registration authority
* the Reference Document class to identify the language(s) used within the document
* the Language Section class of the Naming and Definition region to identify the language used for names and definitions within that section.

The identifier comprises a mandatory language identifier and an optional country identifier, the latter being used to distinguish variations in language use in different countries.
