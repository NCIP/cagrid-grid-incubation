The classification package provides support for classification scheme registry and a mechanism for persisting links to classification schemes and associated items in the metadata repository.

Generalisation relationships between ISO11179 and the cancergrid metadata repository were broken to allow the subsitution of ID references for relationships in the standard.

This is not <originally> intended to hold the entire contents of an external controlled vocabulary but could possibly be extended to do so if required.
