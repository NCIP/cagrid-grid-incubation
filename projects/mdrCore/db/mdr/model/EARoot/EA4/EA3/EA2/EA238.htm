An Administered Item with the Retired status indicates that the Registration Authority has determined the Administered Item is no longer recommended for use in the community that uses this metadata register.  A Retired Administered Item should no longer be used. Such Administered Items are retained in the metadata register archival storage facility for historic reference and research purposes.<br /><br />Retired Administered Items should include a reference to replacement Administered Items when appropriate. Only editorial edits of Retired Administered Items are permitted.
