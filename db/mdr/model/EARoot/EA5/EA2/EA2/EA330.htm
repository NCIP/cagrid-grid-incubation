This transformation implements the generation of a 'standard' ISO document, gathering resources necessary for the export of data elements that reference the Classification_Scheme.

classification_scheme_item_value has been restricted to being any XML content to allow the structure of the classification scheme item entry to be persisted (either as XML or XHTML) in the record.
