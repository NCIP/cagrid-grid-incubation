This diagram describes the correspondence and specifies artefacts necessary for the translation between the ISO11179 standard and its cgMDR implementation.<br /><br />The is11179:value class has been combined with the iso11179:permissible_value class - explicit declaration of permissible values is required by cgMDR.<br /><br />Value meanings are associated seperately from permissible values: an enumerated conceptual domain may be associated with a number of value domains with restricted permissible value sets.  In the interface, the user will only be presented with previously declared value meanings from the selected enumerated conceptual domain.<br />
