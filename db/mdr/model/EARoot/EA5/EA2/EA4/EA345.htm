This diagram describes the detail of the Data Element document, and details its relationships with other documents in the cgMDR implementation of ISO11179.<br /><br />A new administered class 'framed question' is added to support data capture forms.  Essentially the submitter may frame a question to elicit the response of the data element, and give instructions to the completer of the data element.  Typically this question will be used to create data capture forms.  As an administered item the framed question has language attributes and thus the construct can be used to support multilingual data capture instruments.  Provenance surrounding the translation of framed questions may be recorded as reference documentation.<br /><br /><br />
